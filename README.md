# Java-googleMaps

### 자바 응용 프로그램 개발 구글맵연동 프로젝트


- 주요기능

1. 구글맵이 연동이 되었다면 검색창에 원하는 주소를 입력했을때 구글맵 이미지로 실시간으로 보여집니다 
2. (구글맵 연동이 안된것이 아쉬운 프로젝트입니다...)

- 오류사항

1. 구글맵이 유료화가 되면서 실행했을때 구글맵이 나타나지 않는다
2. 실행결과에 URL링크가 서버에서 HTTP 응답코드를 반환해 구글맵이 실행되지 않는다
